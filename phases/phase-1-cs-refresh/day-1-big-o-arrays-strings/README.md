# Day 1 — Big O, Arrays, Strings, Set/Map Optimization

## Goal

By the end of this day, I should be able to explain how to improve an `O(n^2)` brute-force solution into an `O(n)` solution using `Set` or `Map`.

## Learning Objectives

- Explain `O(1)`, `O(n)`, `O(n^2)`, `O(log n)`, `O(n log n)`.
- Understand time complexity vs space complexity.
- Understand array search, insertion, removal, and sorting.
- Know which JavaScript array methods mutate and which return new arrays.
- Understand strings as immutable sequences.
- Recognize palindrome, anagram, and character frequency problems.
- Know when to use `Set`.
- Know when to use `Map`.
- Solve Contains Duplicate, Two Sum, and Valid Anagram.
- Explain trade-offs in English.
