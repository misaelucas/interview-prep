# Phase 1: Computer Science Refresh

Phase 1 is a practical computer science refresh for coding interviews.

The goal is not academic theory. The goal is to recognize what kind of problem is in front of you, choose a reasonable data structure, explain the complexity, and communicate the trade-offs clearly.

## Goals

- Identify problem patterns.
- Choose the right data structure.
- Explain time complexity.
- Explain space complexity.
- Improve brute-force solutions.
- Communicate clearly in interviews.

## Days

- Day 1: Big O, Arrays, Strings, Set/Map Optimization
- Day 2: Hash Tables
- Day 3: Stack, Queue, Linked List
- Day 4: Recursion, Trees, DFS/BFS
- Day 5: Graphs and Review
